# infoset-react-native

Infoset SDK for react-native applications

## Installation

```sh
npm install infoset-react-native
```

## Usage

```js
import InfosetReactNative from "infoset-react-native";

// ...

const result = await InfosetReactNative.multiply(3, 7);
```
